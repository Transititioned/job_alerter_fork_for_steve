# Job Alerter Fork Notes

This repo is a lightweight mirror/runbook for a working Google Apps Script
job-alerting app.

The Apps Script editor remains the working runtime. Git is used for snapshots,
handover notes, and repeatable manual forking.

Start here:

- `GAS_FORKING.md` - manual steps to create a new fork of the app.
- `ENVIRONMENT.md` - script properties, triggers, deployments, sheets, Drive
  folders, and other setup that is not captured by copied `.gs` files.
